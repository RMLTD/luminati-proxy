Automatic method

If you are using outdated version of the proxy manager tool, there should be a notification on the top of this page. Click the Upgrade button, provide your system password (if necessary) and wait while the application is being upgraded and restarted (this might take a while).

If the automatic upgrade failed, please use the manual method described below.

Manual method

To upgrade the proxy manager tool to the latest version, quit the tool and run the following command: 
$ npm install -g luminati-io/luminati-proxy

Note that the source code of the Luminati proxy manager tool is available at GitHub.
